## About The Project

[//]: # ([![Product Name Screen Shot][product-screenshot]]&#40;https://encrypted-tbn0.gstatic.com/images?q=tbn:ANd9GcRo9vRXBMGVaFpMHMNMSHjKS-snQ0MjWZ4HWw&usqp=CAU&#41;)

This project contains all the utilities needed to install all developer tools in a matter of seconds. Backup your local files to the cloud and restore them on a different computer. `All within one SDK`

Here's why:
* Stop wasting a full day of work configuring your new computer
* Allow to easily change between computers

<!-- GETTING STARTED -->
## Getting Started

### Prerequisites

* winget package manager (Windows)
  ```
  https://docs.microsoft.com/en-us/windows/package-manager/winget/#production-recommended
  ```

### Installation


1. Go the to repo Releases
2. Download source code
3. Run `StationSetup.Cli.Exe` you will have 3 options to:
    * Install tools like VSCode, Postman, etc
    * Copy folder to AWS S3 Cloud
    * Copy from AWS S3 cloud
